# NodeJS + ExpressJS REST API for Startup Managing

## a fully functional endpoints for project managers to manage their projects

This project is an backend project that includes RESTFUL API's for project managers to manage their projects. Front-end for this project will be written in flutter and is work in progress. this project was builded to achieve this following goals:

- Ease access for project managers to manage their startup
- Creating and removing multiple startups
- Removing people from starups
- Give roles / permissions to people in the startup to create / update / delete tasks
- Create / Update / Delete tasks for each startup and assign tasks to users that are in startup
